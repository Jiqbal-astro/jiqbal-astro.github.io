Projects

Selected software tools and modeling efforts that support my research in exoplanets, Galactic H I, and space systems.

Exoplanet Transit Simulator

A browser-based simulator that models exoplanet transits and generates synthetic light curves including limb-darkening. Users can adjust orbital parameters, explore multi-planet systems, and compare setups to real exoplanets such as TRAPPIST-1e and Kepler-186f.

GitHub / Live Demo →

Lunar & Planetary Phase Simulator

An interactive tool for visualizing lunar and planetary illumination phases and reflected-light curves. Features real-time Lambertian disk rendering, date-linked lunar phases, adjustable inclination and eccentricity, and export options for PNG and CSV.

GitHub / Live Demo →

Galactic H I Drift-Scan Pipeline

A Python pipeline designed for small-aperture radio telescopes that automates RFI flagging, baseline removal, Gaussian profile fitting, and column-density estimates from 21 cm spectra. The pipeline cross-checks results with the LAB survey for validation.

Transit Timing Variation Tools

Custom scripts that interface with MCMC samplers and N-body integrators to model transit timing variations. These tools are used in collaboration with ÇOMÜ Ulupınar Observatory to search for and characterize non-transiting companions in known exoplanet systems.
